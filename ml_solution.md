# Блок «Машинное обучение» — решение задач 22–24

Источник заданий: PDF с вариантом Sirius. В этом блоке дана модель логистической регрессии с параметрами `w0 = 1`, `w1 = 7`, `w2 = 7`, `p = 1/2`, а также объект `x1 = -0.2`, `x2 = 0.3`. Нужно найти значение сигмоиды, итоговое предсказание, затем посчитать accuracy на `validation_set.csv` и подобрать порог `p`, при котором accuracy максимальна.

## Ручное решение задач 22–23

### 1. Считаем линейную часть

\[
z = w_0 + w_1 x_1 + w_2 x_2
\]

Подставляем значения:

\[
z = 1 + 7 \cdot (-0.2) + 7 \cdot 0.3 = 1 - 1.4 + 2.1 = 1.7
\]

### 3. Предсказание класса

По условию:

\[
\hat{y} = sign(\sigma(z) - p), \quad p = 0.5
\]

\[

\]

**Ответ: 1**

## Что делает скрипт

1. считает ответы на задачи 22–23;
2. читает `validation_set.csv` через `pandas`;
3. по формуле логистической регрессии считает вероятности;
4. считает accuracy при `p = 0.5` с помощью `sklearn.metrics.accuracy_score`;
5. перебирает много значений `p` на отрезке `[0, 1]`;
6. находит значение порога, при котором accuracy максимальна;
7. строит график `accuracy(p)` и сохраняет его как `ml_accuracy_vs_threshold.png`.

В загруженном PDF есть условие и список колонок `x1`, `x2`, `y`, но самого файла `validation_set.csv` в чате нет, поэтому точные ответы на задачи с accuracy и подбором `p` нельзя честно вычислить без исходной таблицы. Скрипт готов и посчитает всё автоматически, когда файл будет рядом.
