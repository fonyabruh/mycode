# Блок «Анализ данных» — решение задач 19–21

Источник заданий: PDF с вариантами Sirius. В этом блоке используются данные из файла `actions.csv` с колонками `ip`, `date`, `action`. Задачи формулируются так: найти дату с максимальным числом активных пользователей, число уникальных пользователей в марте 2023 года и общее число действий `click` в марте 2023 года.

## Что делает решение

Скрипт:

1. считывает `actions.csv` через `pandas`;
2. переводит столбец `date` в тип даты;
3. считает число уникальных пользователей по каждой дате;
4. находит дату, когда пользователей было больше всего;
5. фильтрует строки за март 2023 года;
6. считает число уникальных пользователей в этом месяце;
7. считает суммарное число действий `click` в марте 2023 года;
8. строит график числа активных пользователей по датам и сохраняет его как `analysis_daily_users.png`.

## Логика подсчёта `click`

В столбце `action` хранится строка вида:

- `search`
- `click`
- `search, click, search`

Поэтому для каждой строки мы:
- разбиваем строку по запятым;
- убираем лишние пробелы;
- считаем, сколько раз встретилось слово `click`.

## Что нужно положить рядом со скриптом

- `actions.csv`

## Что выведет программа

- ответ на задачу 19: дата с максимальным числом активных пользователей;
- ответ на задачу 20: число уникальных пользователей в марте 2023 года;
- ответ на задачу 21: число действий `click` в марте 2023 года.

## Примечание

В загруженном PDF есть только условие, но нет самого файла `actions.csv`, поэтому окончательные числовые ответы нельзя честно посчитать без исходных данных. Скрипт полностью готов и даст ответы сразу после запуска на реальном файле.
